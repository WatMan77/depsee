# Project
## How to run
### Development mode

Front-end:
```bash
# Go inside the directory
cd src/client

# Install dependencies (No need after first time)
npm i

# Start development server
npm run dev
```

Back-end:
```bash
# Go inside the directory
cd src/server

# Install dependencies (No need after first time)
npm i

# Start development server
npm run dev
```
